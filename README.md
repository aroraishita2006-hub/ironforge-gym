# IRONFORGE GYM  💪

IRONFORGE GYM is a responsive gym and fitness website developed using HTML, CSS, and JavaScript. The project provides a modern user interface for gym members and visitors with interactive sections and responsive design.

## Features

- Responsive website design
- Home page with navigation menu
- Membership section
- BMI calculator
- Workout and fitness information
- Contact section
- Interactive JavaScript functionality

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Project Purpose

This project was created as a web development practical/project to demonstrate frontend development skills using HTML, CSS, and JavaScript.
